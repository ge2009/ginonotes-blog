# Peninsula：macOS 下的快速切换工具

Peninsula 是一款优雅的 macOS 应用切换工具，它提供了比系统默认 Cmd-Tab 更美观和实用的切换体验。

## 项目信息

GitHub 地址：[Peninsula](https://github.com/Celve/Peninsula)

## 主要特点

- 美观的界面设计
- 流畅的切换动画
- 便捷的操作方式
- 完全替代系统默认切换器

## 使用方法

### 基本操作

Peninsula 使用 ⌘ + Tab 组合键进行应用切换，提供以下操作方式：

1. **切换到下一个应用**
   - 按住 `⌘` 并点击 `Tab`

2. **切换到上一个应用**
   - 按住 `⌘` + `Shift`

3. **退出切换器**
   - 按住 `⌘` 并点击 `Esc`
   - 不会执行任何切换操作

4. **选择应用**
   - 松开 `⌘` 键即可切换到选中的应用
